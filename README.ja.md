## ゲーム紹介

『液体囲碁』は、伝統的な囲碁ゲームの電子的な変種です。本作では、石はもはや格子点に正確に置かれる実体ではなく、ぼやけた柔軟な勢力範囲となり、盤上の任意の位置に置くことができます。

死活判定の論理は伝統的なルールと同じです。すなわち、一団の石が完全に囲まれ、ダメ（空点）がなくなれば死にます。しかし、より自由な着手ルールのもとでは、伝統的なルールでは常識とされていた多くのことが大きく変化します。たとえば、最少三つの石で相手の一子を取ることができる、もともと連続していた一団の石が分断される、シチョウがより難しくなる、などです。たとえ囲碁に熟達したプレイヤーであっても、本作では新しいゲームに出会ったときのように、ルールを探求する楽しさを体験できます。

「見た目は囲碁」の伝統モードのほか、本作では以下のようなルール変種の対応を予定しています。

- 調整可能な盤面サイズ；
- 2〜4 人が同じ盤上で対局；
- 球面盤；
- 縮小円モード（盤面が徐々に縮小し、最終的に一方が盤面を独占する）。

以下の内容は、今後のバージョンで提供予定です。

- 筆モード（対戦する各陣営がリアルタイムで盤上に描く）；
- 雨滴モード（盤上に中立の雨滴が徐々に出現する）；
- 円形、および自由に形を設定できる盤面。
