# 连续围棋（Fluid Weiqi）

## 1. 这是什么
连续围棋是一个受围棋启发的实验性策略游戏。与传统围棋不同，本作基于连续影响场进行对局，而不是纯离散棋子规则。

当前版本用于玩家测试与反馈，不代表最终正式规则。

## 3. 操作说明
- 移动鼠标：预览落子位置
- 鼠标左键：落子
- 按住 Shift 再左键：自由落子（连续模式）
- 不按 Shift 左键：格点落子（吸附到交叉点）

## 4. 当前规则（测试版）
- 回合制，玩家轮流落子
- 棋子会产生影响力分布（高斯密度场）
- 已启用占位检测（已占区域不可再下）
- 已启用连通与提子逻辑
- 已启用边界吸附行为

说明：当前仍是实验规则，和传统围棋并不完全一致。
